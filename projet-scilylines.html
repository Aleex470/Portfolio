Réalisation d'une interface graphique en C# et d'une application web en Symfony pour la réservation de bateau en Italie.

Lors notre deuxième année de BTS SIO SLAM, nous devions réaliser une interface graphique en C# avec Visual Studio 2022, permettant à un administrateur d'ajouter d'une nouvelle liaison en fonction de son secteur, de modifier de la durée d'une liaison, et de supprimer une laision existante.
Afin de rendre possible toutes ces modifications, il a fallu créer une base de données en SQL sur phpMyAdmin avec les informations nécessaire. Ensuite, faire la connexion avec la base de données depuis visual Studio 2022 pour récupérer les données que l'on va ensuite afficher, modifier ou supprimer grâce à certaines fonctions et méthodes.
